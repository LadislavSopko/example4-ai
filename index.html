The solution

SCIP-indexed. Zero install.

xmp4 is a hosted MCP server. 547 popular open-source libraries are pre-indexed with SCIP — the semantic code format Sourcegraph uses. Your AI agent gets type-resolved symbols, call graphs, inheritance chains, and source bodies via 17 MCP tools. No cloning, no grep, no false positives.

Context7 gives you docs. GitMCP gives you file paths. xmp4 gives you the compiler.

Privacy

What we log. What we don't.

✓ WE LOG

  • Aggregate query counts (symbol name, project name, tool name) to grow the library index by demand.
  • Coarse-timestamped tallies to see which libraries are trending up in usage.
  • Standard nginx access logs (IP + UA) kept 7 days for abuse prevention, then purged.

✗ WE DO NOT LOG

  • Contents of your codebase — xmp4 is read-only on OSS libraries we indexed. We never receive your files.
  • Personal identifiers, accounts, or auth tokens — the public endpoint is open-auth.
  • Request bodies beyond the declared tool parameters. MCP protocol scope only.
